# Final Project Showcase

Dokumentasi ini berisi kumpulan aplikasi dengan desain antarmuka modern yang dikembangkan untuk kebutuhan manajemen sekolah dan operasional. Setiap aplikasi memiliki tampilan yang konsisten, bersih, dan mudah digunakan.

---

## Daftar Aplikasi

### 1. Aplikasi Manajemen SPP
Sistem pembayaran SPP yang mempermudah proses administrasi, pencatatan transaksi, dan pengelolaan data siswa. Desain dibuat minimalis dengan fokus pada keterbacaan data.

**Link Final:**  
https://myparkir.my.canva.site/pembayaranspp

**Fitur Utama:**
- Dashboard ringkasan pembayaran
- Tabel data siswa dan status SPP
- Riwayat transaksi
- Tampilan responsif

---

### 2. Aplikasi Manajemen Parkir
Aplikasi pencatatan parkir untuk kendaraan yang masuk dan keluar, dilengkapi laporan dan rekap harian. Desain antarmuka dibuat sederhana dan cepat digunakan.

**Link Final:**  
https://myparkir.my.canva.site/parkir

**Fitur Utama:**
- Input data kendaraan masuk/keluar
- Rekap harian dan total kendaraan
- Tabel transaksi parkir
- Layout bersih dan terstruktur

---

### 3. Aplikasi Manajemen Absensi Siswa
Aplikasi absensi yang membantu penginputan data, rekap kehadiran harian, dan pengelolaan status siswa. Dibangun dengan tampilan minimal modern.

**Link Final:**  
https://myparkir.my.canva.site/absensisiswa

**Fitur Utama:**
- Tabel absensi harian
- Filter berdasarkan tanggal, kelas, dan status
- Ringkasan hadir/izin/alpa
- Navigasi sederhana dan konsisten

---

## Catatan Desain

Semua aplikasi dirancang dengan pendekatan yang sama untuk menjaga konsistensi.

**Gaya Desain:**
- Minimalist UI
- Grid layout rapi
- Warna solid dengan kontras lembut
- Typography clean
- Navigasi ringkas dan intuitif

**Fokus Pengembangan:**
- Kemudahan penggunaan
- Alur kerja sederhana
- Keterbacaan data
- Konsistensi tampilan antar aplikasi
